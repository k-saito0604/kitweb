[現象]
Windows の 「キャッシュされたログオン」 機能を利用できるか？

[説明]
いいえ，GG はWindows の 「キャッシュされたログオン」
機能をサポートしません．パスワードをキャッシュしたい場合は，GGの機能である

クライアント側パスワードキャッシュ
（クライアント側のプロファイルに保存）
サーバ側パスワードキャッシュ （AD上のプロファイルに保存）

を利用して下さい．
