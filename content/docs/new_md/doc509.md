[GGH4.X] GGH4.6 と GGH4.7 / GGH4.8 の関係は？

[説明]
GGH4.6 に "Windows Server 2012 対応" の機能を追加した製品が GGH4.7 で，
GGH4.7 で見つかった不具合を修正した製品が GGH4.8 です．従って "Windows
Server 2012 対応" が増えた点を除き，それらに機能的な差異はありません．
