[Windows/PowerShell] PowerShell
を使用してリモートホストでコマンドを実行する方法は？

[説明]
以下のように Invoke-Command
によってリモートホストでコマンドを実行することが出来ます．

    Invoke-Command <リモートホスト> -ScriptBlock { <コマンド> }
