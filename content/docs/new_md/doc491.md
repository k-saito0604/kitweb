[番号]
技術ノート KGTN 2013111501

[現象]
[GGH4.X] Windows Server 2012
上で動作するGGサーバでは，日本語入力が画面左上に表示される．

[説明]
IMEで文字を入力する時，未確定の文字が画面左上に表示される現象は
（現時点での） 制限事項です．

    On Japanese systems, the IME window may not display in the usual location.
