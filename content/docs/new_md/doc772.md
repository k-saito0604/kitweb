[GGH4.5] SRW とは何か？

[説明]
SRWは，GGサーバに含まれるサポート依頼のための情報収集ツール （Support
Request Wizard） のことです．スタート → プログラム → GraphOn GO-Global 4
→ Tools → Support Request Wizard で起動することが出来ます．
