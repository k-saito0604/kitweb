技術ノート KGTN 2011061601

[現象]
Windows Server 2003 の C:¥Windows¥System32
の下にあるファイルを消したが，暫くすると復活する．

[説明]
この現象は，Windows ファイル保護機能 （Windows File
Protection，略称WFP）
が働いたためです．詳細についてはマイクロソフト社の知識ベース KB222193
をご覧下さい．

<http://support.microsoft.com/kb/222193/ja>
