[Windows/PowerShell] Windows 10 の環境で PowerShell を使いたい．

[説明]
スタート → Windows Power Shell → Windows Power Shell，右クリック，その他
→ タスクバーにピン留めする で Windows Server 2012 R2
と同様にタスクバーから起動することが出来ます．
