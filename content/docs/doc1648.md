Microsoft Authenticode のPFXファイルの入手方法は？

[説明]
Thawte社のコードサイニング認証を利用した場合，証明書は申請に使用したPCに直接インストールされます．PFXファイルを入手するには，
「IE8 → ツール → インターネット オプション → コンテンツ → 証明書 →
個人」 にインストールされた証明書をエクスポートします．
「秘密キーのエクスポート」 および　「証明パスにある証明書の付加」
方法は，以下の画面を参考にして下さい．
