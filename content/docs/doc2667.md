[現象]
[GGH4.X-5.X] セッション毎にGGが使用するメモリの量 （ワーキングセット）
を知りたい．

[説明]
GGのセッション上で動作するプロセスおよび関連するプロセスのメモリの量
（ワーキングセット） の実測値は以下の通りです．

logon.exe 約4.5MB
remoteClip.exe 約9.2MB
pw.exe 約18.5MB
ctfmon.exe 約8.6MB
JpnIME.exe 約7.8MB

なお，この値 （合計すると約50MB） は Windows Server 2012 R2
の環境で，GGにログオンした直後にタスクマネージャで表示された結果の平均値です．
