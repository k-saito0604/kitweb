技術ノート KGTN 2010041506

[現象]
GGサーバの 「プリンタとFAX」 フォルダから，プリンタが消えた．

[説明]
Windows のサービス Print Spooler
が停止したのが原因だと考えられます．このサービスは，印刷処理が過負荷な時に停止することがあり
（ターミナルサービス等でも発生します）
，これが発生するとプリンタの一覧が表示されなくなります．対応は，停止した
Print Spooler サービスを 「開始」 して下さい．
