[GGC4.0] GO-Global Cloud の管理者のユーザIDは？

[説明]
GO-Global Cloud と Apache Tomcat の管理者のユーザIDは admin です．
PostgreSQL の管理者のユーザIDは， Windows のアカウントおよびpostgres
です．パスワードは，それぞれインストール時に指定したパスワードです．
