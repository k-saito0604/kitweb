[現象]
GO-Global for Windows Ver3.2系は，Windows Server 2008 R2 （64ビット）
をサポートするのか？

[説明]
現行バージョンでは Windows Server 2008 R2 （64ビット）
をサポートしませんが，次期メジャーバージョン "GO-Global for Windows
Ver.4"
ではサポートする予定です．なお，リリース時期等については未定です．
