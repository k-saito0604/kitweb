技術ノート KGTN 2011091401

[現象]
[GGW3.2.1] Win32 API の MoveWindow
で表示中のウインドウを移動すると画面上にゴミが残ることがある．

[説明]
この現象は GGW3.2.1 の既知の問題 （制限事項）
で，以下のように移動の直前で非表示にして，移動の直後に表示して下さい．

ShowWindow(hWnd,SW_HIDE);
MoveWindow(hWnd, ... );
ShowWindow(hWnd,SW_SHOW);
