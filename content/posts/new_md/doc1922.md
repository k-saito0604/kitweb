[現象]
[GGH4.X-5.X] セキュアーな環境で Windows Server 2003
でGGサーバーを運用したいが，どのバージョンを使用するのが良いか？

[説明]
現行バージョンで32ビット版 Windows
をサポートしている製品はバージョン4.8ですので，セキュアーな環境で
Windows Server 2003
のGGサーバーを運用する場合は，このバージョン4.8の使用をお勧めします．
