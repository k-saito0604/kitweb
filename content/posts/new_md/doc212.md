[番号]
技術ノート KGTN 2014081501

[現象]
[GGH4.X] GGサーバへのログオン時に，Windows
の認証画面が表示されるようになった．

[説明]
GGのログオン画面でなく Windows
の認証画面が表示される場合は，恐らくスマートカード認証が有効になっていることが考えられます．クラスタマネージャを起動し，

Tools → Host Options ... → Security → Authentication → Smart card
authentication

のチェックを外して下さい．
