[現象]
[GGH4.0] Windows Client の中に Clientlauncher.exe と gg-client.exe
があるが，どちらを使用するのか？

[説明]
Clientlauncher.exe は， Windows Client (Native Client) とGO-Global Cloud
Client のAdobe AIR
に関する部分のインタフェースとしての役割を持ちます．従って， Windows
Client としては gg-client.exe を使用して下さい．
