技術ノート KGTN 2014012902

[現象]
[GGH4.X-5.X] Windows Server 2012 (R2) + GGH4.X/5.X でオプション -dfw 0
を指定すると枠のゴミが残る．

[説明]
この現象はGGサーバのOSが Windows Server 2012 (R2)
の場合に限り発生する現象で，現時点では制限事項（既知の問題）
となります．問題が解消されるまで， -dfw 1 （または -dfw
オプションを指定しない） を指定して下さい．
