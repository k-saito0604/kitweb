[現象]
[GGH5.X] Windows Server 2016 の環境でGGサーバーを運用したい．

[説明]
GGH5.0.2.25297 (LA版) またはそれ以降のビルドが，ホスト環境として Windows
Server 2016 をサポートしております． Windows Server 2012 R2
の環境と同様に，サーバー側のIMEが動作します．
