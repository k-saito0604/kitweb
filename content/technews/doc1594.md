[番号]
技術ノート KGTN 201010401

[現象]
Active Directory (2008)
を使用しているが，移動ユーザープロファイルのフォルダ名に ".V2"
が付いている．

[説明]
クライアントが Windows Server 2008， Windows 7， Windows Vista
の場合，従来の Windows と区別するために， "<Username>.V2"
という名前のプロファイル・フォルダが作成されます．
