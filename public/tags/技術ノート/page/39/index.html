[GGH4.X-5.X/Windows] ADPlus で logon.exe のダンプファイルを採取する手順は？

in tech

2017-05-08

*技術ノート 2017/05/08* [番号] 技術ノート KGTN 2017042603 [現象] [GGH4.X-5.X/Windows] ADPlus で logon.exe のダンプファイルを採取する手順は？ [説明] 技術ノート KGTN#2017042601 または KGTN#2017042602 を参照して， Debugging Tools for Windows をインストールしてください．ダンプの採取は，このツールの中の adplus.exe を使用します． <STEP#1> ダンプファイルを出力するフォルダー C:DUMP を作成します． <STEP#2> 管理者でコマンドプロンプトを開き， adplus.exe があるフォルダー “C:Program Files (x86)Windows Kits10Debuggersx64” へ移動します． <STEP#3> ダンプファイルを採取する logon.exe のプロセスIDを調べます．これには，クラスターマネージャー ・ タスクマネージャー ・ Process Explorer 等を使用してください．下記の例では， 3996 が logon.exe のプロセスIDです． <STEP#4> コマンドプロンプトで “adplus -hang -p -o C:DUMP” を実行します （ハングモードでのダンプファイルの採取） ． <STEP#5> 続けて，コマンドプロンプトで “adplus -crash -p -o C:DUMP” を実行します （クラッシュモードでのダンプファイルの採取） ．
