in tech

2014-01-31

*技術ノート 2014/01/31* [番号] 技術ノート KGTN 2014013001 [現象] [GGH4.X] ActiveXクライアントをダウンロードしたがログオン画面が表示されない．何を調べれば良いか？ [説明] 以下の手順で操作を行って下さい． クライアントPCを再起動 Administrator でログオン GGAXDelete4G.exe を起動 確認ボタンをクリック メモ帳で表示されたファイル（ログ）を保存 … ファイル名 info.txt で保存 削除ボタンをクリック メモ帳で表示されたファイル（ログ）を保存 … ファイル名 del.txt で保存 GGサーバにログオン この手順を行っても問題が解決しない場合は，上記 5) と 7) で保存したファイル （info.txtとdel.txt） をお送り下さい． ファイル KGTN2014013001.pdf [(54 KB)] [kitasp 技術センター, 2014/01/31 14:19]
