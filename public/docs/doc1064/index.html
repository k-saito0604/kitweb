[iPad Client] WiFi環境で iPad Client を使用しているが，たまに数秒間止まった感じになる．　

2012-03-07

*技術ノート

2012/03/07*

[番号] 技術ノート KGTN 2012020101

[現象] [iPad Client] WiFi環境で iPad Client を使用しているが，たまに数秒間止まった感じになる．

[説明] 右クリック操作 （コンテキストメニューの表示） で数秒間止まる現象を，弊社およびGraphOn社で確認しております．現時点では修正のスケジュールは未定ですが，GraphOn 社より新たな情報が届き次第，この技術ノートに反映します．なお，この現象はAndroid Client では発生しません．

(2012-03-07) この問題は 4.1.1.12004 でも発生します．
