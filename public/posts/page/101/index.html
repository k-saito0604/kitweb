in tech

*技術ノート 2015/06/11* [番号] 技術ノート KGTN 2015022503 [現象] [Windows] 物理マシンへ Windows (WS2008R2やWS2012R2) を無人インストールする方法は？ [説明] 以下の手順で，Windows (WS2008R2やWS2012R2) の無人インストールを行うことが出来ます． Windows System Image Manager で応答ファイルを作成する． 応答ファイルを Autounattend.xml というファイル名でUSBディスクのルートに保存する． サーバーに WindowsのインストールDVDと上記のUSBディスクをセットする． サーバーを起動する． https://msdn.microsoft.com/ja-jp/library/hh824950.aspx Windows セットアップの自動化の概要 ファイル KGTN2015022503.pdf [(55.2 KB)] [kitasp 技術センター, 2015/06/11 17:24]
